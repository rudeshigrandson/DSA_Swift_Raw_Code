## Practical Planning Rules

- Weekdays are kept mostly around `1.5 to 2.5 hours` of watch time so you still have room to code.
- Weekends hold the large concept blocks like `OOP`, `Linked List + Stack + Queue + Trees`, and `DP + Graphs`.
- `Videos 20-39` stay on the same day so your OOP concepts are not broken across multiple sessions.
- The goal is not only to watch; after each block, implement the core idea in Swift.
